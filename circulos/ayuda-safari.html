⚠️ Problema con Enlaces en Safari

Safari puede tener problemas para abrir enlaces cuando se accede directamente a archivos HTML desde el sistema de archivos.

✅ Soluciones:

  1. Usar un servidor web local: Abre los archivos desde un servidor web en lugar del sistema de archivos.
  2. Copiar y pegar las URLs: Copia la URL del enlace y pégala en la barra de direcciones.
  3. Usar otro navegador: Chrome o Firefox suelen manejar mejor estos enlaces.
